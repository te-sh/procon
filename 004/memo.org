** No.152

$a + b + c \leq \lfloor L/4 \rfloor$ となるピタゴラス数を数え上げる。

ピタゴラス数の数え上げは

$(a, b, c) = (m^2 - n^2, 2mn, m^2 + n^2)$ (ただし、$m \gt n$, $m$ と $n$ は互いに素、$m - n$ は奇数)

を使う。$2m^2 \leq L/4$ であるから、$m$ は最大でも $10^3$ 強である。
